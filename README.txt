Client folder:
 client.py : começa o programa do cliente. Variaveis padrão, como por exemplo o username e os diretorios têm de ser definidos dentro de client.py
 decrypt_file.py : decifra todos os ficheiros dentro do diretorio "sync", e coloca o contúdo decifrado na pasta "dec" (esta é criada pelo programa caso ainda não exista)
Server folder:
 server.py : começa o servidor responsável por todo o processo (registo, login, ...)
